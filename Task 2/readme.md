# Task 2

**Brief**

For this task, I set up my profiles on GitHub, LinkedIn, and Kaggle as a first step toward building my digital presence as a student. I realized that these platforms are useful for showcasing skills and connecting with people in the tech field.

On GitHub, I created my profile and added a README file with some basic information like my branch and areas I’m interested in learning. I plan to keep uploading my projects there so I can track my progress and gradually improve.

On LinkedIn, I added my education details, including my degree and expected graduation year. At the moment, my profile is quite basic, but I intend to update it regularly as I gain more skills and experience.

On Kaggle, I explored a few beginner-friendly datasets and notebooks. Initially, it felt a bit confusing, but it helped me understand how data and machine learning projects are structured.

Overall, this task made me understand why it’s important to start building a professional profile early during college.
